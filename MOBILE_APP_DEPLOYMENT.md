# Michiel Jewelry Mobile App

## 📱 **React Native Mobile Application**

A luxury jewelry shopping app that complements your web application with native mobile features.

## 🎯 **Features**

### 🏠 **Navigation**
- Tab-based navigation (Home, Products, Gold Prices)
- Smooth transitions between screens
- RTL Arabic support throughout

### 📱 **Product Display**
- Product catalog with images and details
- Weight and pricing information
- Search and filter capabilities
- Real-time price synchronization

### 📱 **Gold Prices**
- Live gold price display
- Real-time updates from web app
- Countdown timer to market close
- USD price conversions

### 📱 **User Experience**
- Mobile-first responsive design
- Arabic RTL typography
- Touch-friendly interfaces
- Professional luxury theme

### 📱 **Technical Features**
- React Native with Expo framework
- Real-time API integration
- Service Worker for offline support
- PWA capabilities
- Cross-platform compatibility

---

## 🚀 **Installation & Usage**

### **For Users**
1. **Web App**: Continue using your browser at `http://localhost:3000`
2. **Mobile App**: 
   - Install Expo Go app on your device
   - Scan QR code or download from app stores

### **For Developers**
1. **Code Structure**: 
   ```
   /mobile-app/
   ├── App.tsx          # Main mobile app
   ├── main.js           # Expo entry point
   ├── package.json       # Dependencies
   └── assets/          # Images and icons
   ```

2. **Development**:
   ```bash
   cd mobile-app
   npm install
   npm start
   ```
