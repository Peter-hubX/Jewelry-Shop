# Michiel Jewelry Mobile - React Native App

## 📱 **Project Overview**
**React Native app for iOS and Android with Expo framework**

## 🎯 **Features Implemented**
- **Navigation**: Stack navigator with RTL Arabic support
- **Product Display**: Product catalog with prices
- **Gold Prices**: Real-time price fetching from web app
- **Arabic UI**: Full RTL support with Cairo font
- **Responsive Design**: Mobile-first approach

## 📁 **Project Structure**
```
MichielJewelryMobile/
├── App.tsx              # Main app component
├── app.json              # Expo configuration
├── package.json           # Dependencies
├── assets/               # Images and icons
└── babel.config.js         # Babel configuration
```
